# Quickstart: Bauhaus UI Redesign

## Overview
This feature redesigns the Sharp Timer App UI to follow Bauhaus principles.

## Running the App
1. Open `Sharp Timer App.xcodeproj` in Xcode.
2. Select the `Sharp Timer App` scheme.
3. Run (Cmd+R).
4. The app will appear in the menu bar with the new Bauhaus design.

## Verifying the Design
1. Click the menu bar icon.
2. Verify the popover uses the new color palette (Red/Blue/Yellow/White/Black) and geometric shapes.
3. Go to Settings (Gear icon).
4. Verify the settings window matches the theme.
5. Quit the app (Cmd+Q or via menu).
6. Verify the quit confirmation window matches the theme.

## Troubleshooting
- **Colors look wrong**: Ensure your system is not forcing a high-contrast mode that overrides app colors (though the app should handle standard Dark Mode by forcing Light).
- **Layout broken**: Resize the text in System Settings > Displays to see if the layout adapts.
