# Quick Start: Mode Auto-Start Setting

**Date**: 2025-11-21

## Overview

The mode auto-start setting allows users to control whether changing the timer mode automatically starts the timer or leaves it paused.

## How to Use

1. **Access Settings**: Click the gear icon in the menu bar timer interface to open settings.

2. **Locate the Setting**: Find the "Auto-start timer on mode change" toggle in the settings panel.

3. **Configure Behavior**:
   - **Enabled (checked)**: When you change timer modes, the timer will automatically start if it was paused.
   - **Disabled (unchecked)**: When you change timer modes, the timer will remain paused.

4. **Test the Setting**: Change timer modes using the segmented control to observe the behavior.

## Expected Behavior

- **Timer Idle**: Changing mode starts the timer regardless of setting.
- **Timer Paused**: Changing mode starts the timer only if auto-start is enabled.
- **Timer Running**: Changing mode keeps the timer running in the new mode.

## Persistence

The setting is automatically saved and restored when you restart the app.
